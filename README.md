# 🌟 尋找地圖大冒險 (Map Adventure)

這是一個專為國小三、四年級學生設計的 AI 互動英語學習遊戲。

### 🎯 教學目標
* 練習方位介係詞 (on, under, behind, next to, in)。
* 培養完整句型寫作習慣 (It is... / The map is...)。
* 訓練聽力理解與邏輯判斷。

### 🤖 AI 回饋邏輯
本遊戲內建精準的偵測邏輯：
1. **防止作弊**：攔截題目中的 `or` 字眼，強制學生進行選擇。
2. **句型引導**：針對不同關卡強制要求特定主詞。
3. **即時建議**：根據語法錯誤（如 is 放錯位置、缺少主詞）提供視覺化回饋。

### 🛠 使用技術
* HTML5 / CSS3 (Responsive Design)
* Vanilla JavaScript
* Web Speech API (TTS 語音合成)
